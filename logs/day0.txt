

### ✅ AI Tools List

1. *Traycer AI*

   * *Correct Spelling*: Traycer
   * *Description*: An AI-powered coding assistant that plans, implements, and validates code changes. ([GitHub][1], [traycer.ai][2])

2. *Cline*

   * *Correct Spelling*: Cline
   * *Description*: An open-source AI coding assistant with dual Plan/Act modes, terminal execution, and Model Context Protocol (MCP) for VS Code. ([Cline][3])

3. *Windsurf*

   * *Correct Spelling*: Windsurf
   * *Description*: Formerly known as Codeium, Windsurf is an AI-powered code editor designed to enhance developer productivity. ([Windsurf][4])

4. *LMarena*

   * *Correct Spelling*: LMarena
   * *Description*: Also known as Copilot Arena, it's an open-source AI coding assistant providing paired autocomplete completions from different LLMs. ([Axios][5], [GitHub][1])

5. *Grok*

   * *Correct Spelling*: Grok
   * *Description*: An AI assistant developed by xAI, offering advanced capabilities in reasoning, coding, and visual processing. ([xAI][6])

6. *DeepSeek Coder*

   * *Correct Spelling*: DeepSeek Coder
   * *Description*: A series of code language models trained from scratch, optimized for code completion and understanding. ([DeepSeek Coder][7])

7. *OpenRouter.ai*

   * *Correct Spelling*: OpenRouter.ai
   * *Description*: A unified interface for accessing various large language models (LLMs) with competitive pricing and uptime. ([techradar.com][8], [OpenRouter][9])

8. *Gemini Code Assist*

   * *Correct Spelling*: Gemini Code Assist
   * *Description*: Google's AI coding assistant that completes code as you write and generates entire code blocks or functions on demand. ([Google Cloud][10], [Google Cloud][11])
